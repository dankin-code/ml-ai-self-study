# Machine Learning and AI Projects

## This repository hosts personal projects in the ML/AI space where I develop my skills as well as showcase new discoveries.

## What I Want to Accomplish

My goal is to polish my Machine Learning and AI skills by following an intensive, structured learning path to being able to speak fluently about statistics, classical machine learning, and modern deep learning — backed by a portfolio of eight projects I built myself.

### The Three-Book Strategy

The plan braids three books together by theme, so each concept is hit three ways:

- **Practical Statistics for Data Scientists** — the *why*: statistical foundations, sampling, experiments, and the math intuition behind models.
- **Machine Learning with PyTorch and Scikit-Learn** — the *how*: the core technical spine — classical ML in scikit-learn, then deep learning in PyTorch, built up from first principles.
- **Python Machine Learning by Example** — the *do*: real, end-to-end projects (recommenders, ad-click prediction, stock prediction, image and text models) that turn theory into a portfolio.

### The Roadmap

| Section | Focus | Milestone |
|------|-------|-----------|
| **1 — Foundations** | Environment setup, EDA, sampling & distributions, experiments & significance, first classifiers (perceptron/Adaline) from scratch | Explain bias, variance, sampling distributions, and p-values without notes; a classifier implemented from scratch |
| **2 — Classical ML mastery** | Preprocessing pipelines, dimensionality reduction (PCA/LDA), cross-validation & tuning, trees, forests, boosting, ensembles, regression, SVMs, clustering | Take a raw tabular dataset to a tuned, honestly-evaluated model — and defend every choice |
| **3 — Deep learning** | Neural nets & backprop from scratch, PyTorch fluency, CNNs for images, RNNs/LSTMs for sequences, NLP & embeddings | Design, train, debug, and evaluate a neural network in PyTorch for image and text data |
| **4 — Modern AI & capstone** | Transformers & attention (BERT/GPT), GANs, CLIP, graph NNs, reinforcement learning, then a 3-day capstone build and hard interview prep | A polished capstone plus seven supporting projects, interview-ready |

### Study Rhythm per section (10 hours)

1. **Theory & reading** (2.5 hrs) — read the day's chapters, take structured notes.
2. **Guided build** (3.0 hrs) — code along with the book examples; break things on purpose, then fix them.
3. **Applied / project** (2.5 hrs) — apply the technique to my own dataset or push the week's project forward; one working artifact per day.
4. **Review & interview prep** (1.5 hrs) — flashcards, explain-aloud, coding drills, journal.

### Target Project Portfolio

By end of the rhythm, eight projects live in this space — evidence, not certificates:

1. **Movie recommender (Naive Bayes)** — classification fundamentals and probability
2. **Ad click-through prediction** — trees, random forests, XGBoost, logistic regression at scale
3. **Stock-price regression** — regression modeling, residual diagnostics, regularization
4. **Face recognition + clustering** — SVMs, multiclass classification, unsupervised learning
5. **Fashion image CNN** — convolutional networks, data augmentation, transfer learning
6. **Sentiment + text generation (RNN)** — sequence modeling, LSTMs, text data
7. **GAN + CLIP image search** — generative modeling and modern multimodal AI
8. **Capstone (my choice)** — end-to-end ownership: framing, building, evaluating, and communicating a real solution
